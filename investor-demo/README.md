## Why these three

Investors ask two questions about AI product companies:

1. **Is the output actually good?** → Skeptic outputs answer this. Both
   read as strategic diagnoses, not marketing fluff. Both name a
   specific pillar. Both propose deliverable-grade fixes.

2. **Why do people keep paying?** → Coach output answers this. Ongoing
   focused actions grounded in the client's own memory create a retention
   surface no one-time deliverable can match.

## The delivery cost

- Skeptic: ~$0.09-$0.15 per output
- Coach: ~$0.05-$0.08 per weekly check-in
- Per Premium Snapshot delivery ($499 sale): ~$1-3 in agent costs
- Gross margin: 99%+

## The moat visible in these outputs

Same agent, same tool set, same methodology — different diagnoses per
client because retrieval is per-tenant. That's the switching cost:
the memory lives with TOJ, not with a competitor.
